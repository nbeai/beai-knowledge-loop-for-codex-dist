BEAI Knowledge Loop for Codex는 Codex가 작업 전 과거 판단을 찾고, 작업 후 review-only 지식 후보를 남기게 하는 검토형 작업 기억 루프입니다.

무엇이 다른가

LLM Wiki가 AI가 읽는 도서관이라면, Knowledge Loop는 AI 직원의 업무일지와 인수인계 체계에 가깝습니다.

  • 작업 전: 과거 판단과 근거를 검색합니다.
  • 작업 중: 기존 결정, 함정, 위험 경계를 참고합니다.
  • 작업 후: review-only 지식 후보를 남깁니다.
  • 승인 전: durable memory나 외부 행동으로 승격하지 않습니다.

왜 필요한가

강한 AI일수록 더 많은 판단을 남깁니다. 그 판단이 사람의 검토 없이 조직의 기준이 되는 순간, 편리함은 위험으로 바뀝니다.

BEAI Knowledge Loop는 AI가 덜 헤매게 만들면서도 사람이 판단권을 잃지 않게 하는 쪽에 초점을 둡니다.

현재 상태

현재 패키지는 Codex용 public preview / team install candidate입니다. 고객용 상용 설치 프로그램이나 자동 메모리 제품으로 선언하지 않습니다.

포함된 핵심 구성은 Skill, MCP Tool, local JSON store입니다.
